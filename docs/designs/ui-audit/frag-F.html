개선안 — 무엇이 바뀌는가

  • 사이드바 상단 모드 탭: 데스크탑에서도 Console / Knowledge / Chat 3-tab이 항상 보이도록 sidebar 헤더 아래에 고정. 현재 경로의 모드가 활성. ① 해결.
  • 빈 상태 capability grid: 인사말 아래 4셀(📚지식 · 🧠메모리 · 🔧도구 · 📁프로젝트) 진입점. 각 셀이 클릭 시 해당 도메인으로 점프 + 첫 메시지 템플릿 자동 주입. ② 해결.
  • 인터뷰 자유입력 접기: 항상 보이던 textarea를 <details> "추가 설명 (선택)" 토글로 강등. 단답 문항은 시각적으로 단순화, 다답/주관식만 노출. ③ 해결.
  • 도구 승인 3-옵션: [거부] [1회 승인] [세션 신뢰]. 세션 신뢰 선택 시 같은 세션 동안 동일 도구 재승인 생략. 카드에 호출 명령 preview를 보여줘 맥락 제공. ④ 해결.
  • 지식 저장 pill: text-only 링크 → 테두리 있는 작은 pill 버튼으로 승격. 메타 라인과 분리되어 한 눈에 보임. ⑤ 해소.
