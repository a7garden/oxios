개선안 — 무엇이 바뀌는가

  • 디바운스 정리: `useEffect` + cleanup으로 디바운스, 또는 라이브러리(use-debounce) 도입. 언마운트 시 미해결 타이머 clear. ① 해결.
  • 액션 센터 스트립: 실패/오류 > 0이면 헤더 하단에 풀폭 알림. success는 일반 메트릭 위치 유지. ② 해결.
  • Canvas 첫 진입 안내: 좌상단에 범례(상태 색·A2A 화살표) + 좌하단 "노드 클릭 → 상세" 큐 추가. 그래프 모드 인지 비용↓. ③⑦ 해결.
  • 툴바 재배치: view 토글을 좌측, sort/refresh는 우측 단일 그룹. 모바일(< md)에서는 view 토글만 1행, 나머지 2행. ④ 해결.
  • 필터 칩 라벨 한국어 + 사람 친화적: `search` → "이름", `status` → "상태" 등 매핑 테이블. ⑤ 해결.
  • 테이블 행 affordance: name 셀에 hover 시 row 강조 + status 색 띠는 이미 있음 → 정렬 가능 컬럼 헤더에 명확한 화살표 + 행 클릭 시 detail panel 슬라이드인 안내.
