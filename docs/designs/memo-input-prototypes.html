빠른 메모 입력 — 4가지 방향 프로토타입

동일한 캡처 인박스를 보여주되, 입력 메커니즘만 바꿔 느낌을 비교합니다. 실제 타이핑 + Enter로 캡처해 보세요. 데이터 모델(Chat.md 체크리스트)과 백엔드 훅은 그대로 유지하는 전제입니다.

트레이드오프

      모든 방향은 동일한 데이터 계층(chat_append / chat_delete / checklist_add / journal_add)을 재사용합니다. 라인 포맷 - [ ] \`HH:MM\` text와 msgHash 기반 삭제는 변경하지 않습니다. 최상위 피드(03) 선택 인덱스 매핑은 실구현 시 별도 처리 필요합니다.
