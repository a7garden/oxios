## Files Changed
- `channels/oxios-web/src/routes/workspace.rs` — Replaced simple SkillSummary with full RFC-009 §5.1 response shape. New `skill_entry_to_json()` produces the complete skill JSON with requirements, missing, install, config_checks, status, source, etc. Added `handle_skill_enable`, `handle_skill_disable`, `handle_skill_content` handlers. Uses `SkillManager`-backed ExtensionApi methods (`list_skills_entries`, `get_skill_entry`, `enable_skill`, `disable_skill`).
- `channels/oxios-web/src/routes/resources.rs` — All `/api/programs/*` and `/api/host-tools` endpoints now return deprecation headers (`deprecation: true`, `Link: <successor>`). Programs list/get delegate to skill data. Install/uninstall return 410 Gone. Enable/disable delegate to skill enable/disable. Host-tools returns empty array with deprecation notice. Added `DeprecationHeaders` helper struct.
- `channels/oxios-web/src/routes/mod.rs` — Added 3 new skill routes: `POST /api/skills/{name}/enable`, `POST /api/skills/{name}/disable`, `GET /api/skills/{name}/content`. Exported new handler functions.
